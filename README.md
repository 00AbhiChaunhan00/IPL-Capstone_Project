# 📌 Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on the **Indian Premier League (IPL) 2022** dataset to uncover meaningful insights into team performances, player statistics, toss decisions, venue impact, and overall match trends.

The primary objective is to demonstrate a complete data analysis workflow—from data cleaning to business insight generation—using Python.

---

# 🎯 Project Objectives

✔ Analyze IPL 2022 match data

✔ Identify the best-performing teams

✔ Understand toss-winning patterns

✔ Analyze venue-wise performance

✔ Explore Player of the Match statistics

✔ Compare team performances

✔ Generate meaningful business insights

✔ Visualize data using various charts

---

# 📂 Dataset Information

**Dataset:** `IPL.csv`

The dataset contains match-level information such as:

| Feature | Description |
|---------|-------------|
| 📅 Match Date | Date of Match |
| 🏏 Team 1 & Team 2 | Competing Teams |
| 🏟 Venue | Match Venue |
| 🎯 Toss Winner | Team Winning Toss |
| 🪙 Toss Decision | Bat / Field |
| 🏆 Match Winner | Winning Team |
| ⭐ Player of the Match | Best Performer |
| 📈 Win by Runs | Winning Margin |
| 🤝 Win by Wickets | Winning Margin |
| 🏁 Stage | League / Playoffs |
