# 🏏 IPL 2022 Data Analysis Capstone Project

# 📌 Project Overview

This project analyzes the **Indian Premier League (IPL)** dataset to uncover valuable insights about team performance, player statistics, toss decisions, venues, match winners, and other important trends.

The project demonstrates practical skills in:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insight Generation
- Python for Data Analytics

---

# 🎯 Objectives

The main objectives of this project are:

- Analyze IPL 2022 match data
- Find the best performing teams
- Understand toss winning patterns
- Analyze venue-wise performance
- Explore player performances
- Generate meaningful visual insights
- Practice real-world data analysis using Python

---

# 📂 Dataset

Dataset Used:

**IPL.csv**

The dataset contains match-level information including:

- Match Date
- Teams
- Venue
- Stage
- Toss Winner
- Toss Decision
- Match Winner
- Player of the Match
- Winning Margin
- Runs
- Wickets
- and many more attributes.

# 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/your-username/IPL-Capstone-Project.git
```

Go to project folder

```bash
cd IPL-Capstone-Project
```

Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

Open Jupyter Notebook

```bash
jupyter notebook
```

Run

```
IPL_Capstone_Project.ipynb
```
